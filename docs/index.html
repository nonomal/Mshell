桌面端与 Android 客户端协同

同一套会话、片段和密钥数据，覆盖日常维护与移动应急场景

Windows 桌面端

完整的服务器管理工作台

适合长期使用和重度操作：多标签终端、分屏、SFTP 文件管理、在线编辑、端口转发、服务器监控、命令历史、AI 助手和完整设置中心。

  • SSH / RDP / VNC 会话管理
  • SFTP 上传、下载、权限、在线预览与编辑
  • 命令片段、快捷命令、历史命令与智能补全
  • SSH 密钥管理、会话锁定、加密备份和同步
Android 客户端

移动端 SSH 与数据管理

适合手机上快速连接、复制或执行片段、维护 SSH 密钥，并继续使用桌面端同步过来的会话信息、分类和服务器管理字段。

  • 新增、编辑、删除 SSH 会话和分类
  • 命令片段分类、复制和在当前终端中执行
  • SSH 密钥新增、编辑和本地文件导入
  • PIN / 系统生物识别锁定保护

同步数据

支持桌面端备份 JSON、.mshell 加密备份、GitHub Gist 和 GitLab Snippet 同步 JSON。会话、分组、命令片段、SSH 密钥和关键设置可跨端导入。

长期同步配置

Android 端可保存同步 URL、Token 和同步加密密码，支持“同步”“下载到本地”“上传本地”，避免每次只做一次性导入。

安全边界

备份和同步数据支持加密密码，移动端可显示/隐藏 Token 与密码以减少输入错误；本地安全锁可优先调用系统生物识别。

核心功能

从连接、文件、命令到同步备份，覆盖服务器维护的常用工作流

SSH 会话管理

保存服务器、账号、端口、认证方式、服务商、地区、到期时间和费用信息，支持分组、搜索、排序和快速连接。

多标签与分屏终端

桌面端支持多连接标签、垂直分屏和四宫格分屏，适合同时维护多台服务器或对比执行结果。

SFTP 文件管理

内置文件传输面板，支持上传下载、批量任务、拖拽、权限设置、在线预览和文本编辑。

命令片段与快捷命令

沉淀常用命令、脚本和多行片段，支持分类、标签、变量替换，并可在当前终端会话中快速调用。

智能输入与命令历史

提供命令历史、搜索、收藏、路径补全、命令补全和终端内联建议，减少重复输入。

SSH 密钥管理

管理 RSA、ED25519、ECDSA 密钥，支持生成、导入、编辑、指纹显示和会话绑定。

安全锁与加密存储

桌面端支持会话锁定和凭据加密存储；Android 端支持 PIN 和系统生物识别锁定。

服务器监控与统计

查看 CPU、内存、磁盘、网络状态，并记录连接时长、流量、命令频率和服务器费用信息。

端口转发、跳板与代理

支持本地、远程和动态端口转发，支持跳板机和代理配置，方便访问内网服务。

RDP / VNC 会话

桌面端除 SSH 外，也可管理 RDP 和 VNC 会话，适合把远程连接统一整理到一个入口。

备份、恢复与远程同步

支持普通 JSON、加密 .mshell 备份、加密同步 JSON，并可通过 GitHub Gist 或 GitLab Snippet 同步。

AI 助手与审计日志

桌面端提供终端 AI 助手、命令解释、操作日志和审计能力，便于排查问题和复盘操作。

系统要求与说明

  • Windows 桌面端：Windows 10 或更高版本，建议 4GB RAM 以上
  • Android 客户端：Android 7.0 或更高版本，支持本地数据和远程同步配置
  • 网络：SSH、SFTP、远程同步和更新检查需要可访问对应服务器或 Git 服务
  • 安全：加密备份和同步请妥善保存密码，密码丢失后无法解密备份数据

快速开始

先在桌面端建立完整数据，再通过备份或远程同步带到 Android 端

01

安装桌面端

下载 Windows 安装包，创建 SSH 会话、导入密钥，并设置服务商、地区、到期时间等服务器信息。

02

整理常用数据

建立会话分组、命令片段分类、SSH 密钥和快捷命令，让日常维护操作可以被复用。

03

配置同步备份

使用加密备份或 GitHub/GitLab 远程同步，保存同步 URL、Token 和同步加密密码。

04

接入 Android

在手机端导入或下载同步数据，启用 PIN/生物识别锁定，即可移动连接服务器并调用片段。
