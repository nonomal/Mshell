Core Features

围绕 SSH 维护场景设计，而不是只做一个终端窗口

从连接、输入、文件、脚本到备份迁移，尽量减少重复操作和低价值切换。

SSH

会话管理

保存主机、端口、账号、认证方式、分组、服务商、地区、费用和到期时间，支持搜索、排序和快速连接。

TAB

多标签与分屏终端

支持多会话标签、垂直分屏和四宫格分屏，适合多台服务器并行维护、对比输出和批量排查。

SFTP

文件传输工作台

SFTP 支持上传、下载、队列、暂停恢复、同名处理、权限修改、在线预览与文本编辑。

CMD

命令片段与快捷命令

沉淀高频命令和多行片段，支持分类、变量、复制、发送到指定终端和运行前确认。

KEY

SSH 密钥管理

支持密钥生成、导入、编辑、指纹显示、会话绑定，并可配合脚本完成公钥部署和登录加固。

AI

AI 助手与命令智能

为命令解释、错误分析、补全建议和常见维护操作提供辅助，相关智能能力可在设置中独立开关。

LOG

日志与审计

提供系统日志、审计日志和操作记录能力，日志开关可按需启用，避免不必要的内存和存储消耗。

UI

外观与终端背景

支持现代、简洁、复古等外观风格，终端可设置全局或单会话背景图与透明度，兼顾可读性和个性化。

Workflow

一次维护任务，从连接到收尾都能在同一处完成

MShell 的目标不是替代每一个专业工具，而是把 SSH 日常维护中最常用的动作串成稳定、可复用的流程。

01

建立连接

从会话列表快速连接，按分组、环境、服务商或到期信息管理服务器。

02

执行维护

调用命令片段、历史命令、智能补全或懒人脚本，减少重复粘贴和误操作。

03

传输文件

在 SFTP 面板完成上传下载、权限修改、在线预览和编辑，传输队列集中查看。

04

同步迁移

会话、分组、片段、密钥和关键设置可通过加密备份或远程同步迁移到其他设备。

Lazy Scripts

懒人脚本：把常见服务器操作沉淀成可复用脚本文件

脚本不是简单的一段粘贴命令，而是可部署到远程主机的通用脚本文件。运行前支持选择目标终端与二次确认，减少把脚本发错服务器的风险。

环境初始化 SSH 加固 防火墙端口 常用工具安装 用户管理 磁盘排查

Sync & Security

同步、备份和安全边界清晰可控

适合多设备使用，也适合在重装系统、换电脑或临时使用 Android 端时快速恢复工作环境。

完整可迁移备份

支持普通 JSON、加密 `.mshell` 备份、同步 JSON 导入导出。会话、分组、命令片段、SSH 密钥和关键设置可作为整体迁移。

GitHub / GitLab 远程同步

支持 GitHub Gist 和 GitLab Snippet，同步配置包含 URL、Token 和同步加密密码。Token 建议使用最小权限。

安全锁与敏感信息保护

桌面端支持会话锁定与凭据保护，Android 端支持 PIN 和系统生物识别。加密备份密码丢失后无法恢复，请妥善保存。

  • 命令片段分类、复制和执行
  • SSH 密钥导入和管理
  • 远程同步、PIN 和生物识别

Quick Start

快速开始

1

创建 SSH 会话

填写主机、端口、账号和认证方式。密码登录与密钥登录按选择独立使用，避免凭据混用。

2

整理命令和脚本

把常用命令放入命令片段，把复杂维护操作沉淀为懒人脚本，运行前确认目标终端。

3

配置同步备份

导出加密备份或配置 GitHub/GitLab 远程同步，妥善保存同步密码和 Token。

4

接入 Android

在手机端导入或下载同步数据，开启 PIN 或生物识别锁，处理移动场景下的临时维护。
